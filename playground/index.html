PQCrypto Playground

Interactive WebAssembly exploration of NIST-standardized Post-Quantum Cryptographic algorithms.

ML-KEM-768 (Module-Lattice Key Encapsulation Mechanism)

Standardized in FIPS 203. ML-KEM-768 is a lattice-based KEM designed for general-purpose key establishment, corresponding to Category 3 NIST security level (equivalent to AES-192 in hardness).

Public Key size 1,184 B
Secret Key size 2,400 B
Ciphertext size 1,088 B
Shared Secret size 32 B

Decapsulation

Decrypt the ciphertext using your secret key to retrieve the shared secret.

ML-DSA-65 (Module-Lattice Digital Signature Algorithm)

Standardized in FIPS 204. ML-DSA-65 is a lattice-based digital signature algorithm based on Dilithium, corresponding to Category 3 NIST security level (comparable to SHA-384 / AES-192 in security robustness).

Public Key size 1,952 B
Secret Key size 2,912 B
Signature size 6,688 B

SLH-DSA-SHA2-128s (Stateless Hash-Based Digital Signature)

Standardized in FIPS 205. SLH-DSA (SPHINCS+) is a stateless hash-based digital signature scheme. The "SHA2-128s" parameter set targets NIST security level 1 (equivalent to AES-128) optimized for small signatures.

Public Key size 32 B
Secret Key size 64 B
Signature size 7,836 B
Note: Hash-based signature operations (especially signing) are highly CPU-intensive and may take a few seconds to complete.
